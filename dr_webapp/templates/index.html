DRCNNRB Architecture

CNN with 3 Residual Blocks (ConvBlock + 2× IdentityBlock) and skip connections to prevent vanishing gradients. 26.5M parameters total.

🎯

CBAM Attention

Convolutional Block Attention Module applied after each ResBlock. Channel attention recalibrates feature responses; spatial attention highlights lesion regions.

🔬

Preprocessing Pipeline

Circle crop → Gaussian median subtraction → Gamma correction (γ=1.2) → CLAHE adaptive histogram equalisation. All images normalised to 256×256.

⚖️

Class Imbalance Handling

WeightedRandomSampler + Focal Loss (γ=2.0) + class-weighted loss. Proliferate_DR augmented ×7 from only 708 samples to 4,956.

📈

Training Results

98.29% accuracy on 35,126 test images. Surpasses the paper baseline of 96.23%. Macro F1 = 0.9734. Proliferate_DR recall = 99.58%.

⚡

TTA Inference

Test-Time Augmentation averages predictions across 5 views (original, h-flip, v-flip, zoom, 90° rotate) for robust and reliable classification.
